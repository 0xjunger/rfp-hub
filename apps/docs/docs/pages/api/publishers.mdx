# Publishers

Publishers are API key holders that can create, update, and import opportunities. Publisher management is admin-only.

All endpoints require admin authentication via the `X-API-Key` header.

---

## Create Publisher

```
POST /api/v1/publishers
```

Creates a new publisher and returns a **one-time API key**. This is the only time the key is shown — store it securely.

### Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | string | Yes | Publisher name |
| `slug` | string | No | URL-friendly identifier (auto-generated from name if omitted) |
| `contactEmail` | string | Yes | Contact email |
| `websiteUrl` | string | No | Publisher website |
| `logoUrl` | string | No | Publisher logo |
| `isVerified` | boolean | No | Whether publisher is verified (default: `false`) |

### Response (201)

```json
{
  "publisher": {
    "id": "018e1234-...",
    "name": "Example DAO",
    "slug": "example-dao",
    "contactEmail": "grants@example.com",
    "websiteUrl": "https://example.com",
    "logoUrl": null,
    "isVerified": false,
    "createdAt": "2026-01-01T00:00:00.000Z",
    "updatedAt": "2026-01-01T00:00:00.000Z"
  },
  "apiKey": "rfphub_a1b2c3..."
}
```

### Example

```bash
curl -X POST 'https://rfp-hub-api.fly.dev/api/v1/publishers' \
  -H 'X-API-Key: rfphub_admin_key...' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Example DAO",
    "contactEmail": "grants@example.com",
    "websiteUrl": "https://example.com"
  }'
```

## List Publishers

```
GET /api/v1/publishers
```

Returns all publishers (without API key hashes).

### Response

```json
{
  "data": [
    {
      "id": "018e1234-...",
      "name": "Example DAO",
      "slug": "example-dao",
      "contactEmail": "grants@example.com",
      "websiteUrl": "https://example.com",
      "logoUrl": null,
      "isVerified": true,
      "createdAt": "2026-01-01T00:00:00.000Z",
      "updatedAt": "2026-01-01T00:00:00.000Z"
    }
  ]
}
```

## Get Publisher

```
GET /api/v1/publishers/:id
```

Returns a single publisher by UUID. Returns `404` if not found.

### Parameters

| Parameter | Type | Description |
|---|---|---|
| `id` | UUID | Publisher ID (path parameter) |

---

## Rotate API Key

```
POST /api/v1/publishers/:id/rotate-key
```

Generates a new API key for a publisher. The **old key is immediately invalidated**.

### Parameters

| Parameter | Type | Description |
|---|---|---|
| `id` | UUID | Publisher ID (path parameter) |

### Response

```json
{
  "apiKey": "rfphub_d4e5f6...",
  "message": "Key rotated. Old key is now invalid."
}
```

---

## Revoke Publisher

```
DELETE /api/v1/publishers/:id/revoke
```

Revokes a publisher's access. Sets `isVerified` to `false`.

### Parameters

| Parameter | Type | Description |
|---|---|---|
| `id` | UUID | Publisher ID (path parameter) |

### Response

```json
{
  "message": "Publisher access revoked."
}
```

All publisher operations are recorded in the [audit log](/api/admin#audit-log).
